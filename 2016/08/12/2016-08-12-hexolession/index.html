Windows下Hexo部署站点总结

终于下决心弄弄blog了，以后尽量常更新吧

  1. 安装nodejs windows下64位nodejs
  2. 安装hexo npm install hexo -g
  3. 进入博客文件夹 cd blog
  4. 初始化博客 hexo init

  5. 设置一些模块 如

    1. hexo-generator-feed 生成atom

    2. hexo-generator-sitemap 生成sitemap文件

      具体查询相关文档

  6. 新增文章
    hexo new "文章名"

  7. 多个tags设置方式[tag1,tag2]

  8. 文章分类 md文件中

    categories: 分类名

  9. 发布 _config.yml 中设置

     deploy:  
    type: git  
repo: git@github.com:自己的git地址.git  
   branch: master

    注意格式
  10. 生成及发布
    hexo generate --deploy
  11. 相关命令
    1. hexo clean 清除缓存
    2. hexo g 或 hexo generate 生成静态文件
    3. hexo server 或hexo s 启动本地服务器预览（localhost:4000）
    4. hexo server -p 5000 更改本地服务器端口
    5. hexo deploy 发布到github (需在_config.yml中配置好相关信息)
    6. 快捷方式 生成后部署 hexo generate –deploy 或 hexo deploy –generate

一定注头部格式的空格，好多次生成不出来就发现是一个空格导致
